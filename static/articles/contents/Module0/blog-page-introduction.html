导航与查找文章

 

1. 🔍 快速搜索

 

  • 在页面顶部的搜索框中输入关键词，系统将实时筛选文章的标题、摘要、分类和标签，立即展示匹配的结果。

2. 📚 侧边栏筛选

 

侧边栏是您快速浏览和筛选文章的利器：

  • 分类列表：点击任意分类名称（例如 "Note" 或 "Examples"），主列表将只显示该分类下的文章。
  • 标签云：点击任何标签（例如 "CG" 或 "Guides"），主列表将只显示包含该标签的文章。
  • 重置筛选：点击“ALL”分类，即可清除所有分类和标签筛选，显示全部文章。

个性化定制您的界面

 

页面的右上角有一个设置图标，点击后可展开定制菜单：

 

1. 🌈 主题色定制

 

  • 在“主题色”区域，拖动色调滑块或直接输入 0 到 360 之间的数值，即可实时改变界面的主要颜色。
  • 若想恢复默认蓝色调，点击“重置”按钮即可。

2. 🌄 背景模式切换

 

  • Banner 模式：保留顶部横幅图片，内容区在其下方。
  • Fullscreen 模式：将背景图扩展至全屏，内容区向上覆盖。
  • Solid Color 模式：使用纯色背景（通常是深色或浅色），适用于追求简洁和极简主义的读者。

3. 🌙 白昼/黑夜模式

 

  • 选择 Light 强制切换到亮色主题。
  • 选择 Dark 强制切换到暗色主题。
  • 选择 System 让网站外观与您操作系统的颜色偏好同步。

⬆️ 提升阅读效率

 

  • 滚动到列表底部：当您向下滚动超过第一篇文章后，页面右下角将出现一个“回到顶部”箭头图标。
  • 一键返回：点击该图标，页面将平滑滚动至最顶部，方便您快速查看导航或重新开始。
